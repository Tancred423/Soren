# Factorial Numbers With Recursion

## Task

Design a program to calculate the factorial of a number with recursion.

- 0! = 1
- 1! = 1
- 2! = 2 \* 1
- 3! = 3 \* 2 \* 1
- N! = N \* (N - 1) \* (N - 2) ... \* 3 \* 2 \* 1

## Restrictions

- Use recursion
